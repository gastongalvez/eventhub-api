#  EventHub API 

Requisitos para ejecutarlo

Antes de abrir el proyecto, asegurarse de tener instalado:

- ✅ [Visual Studio 2022](https://visualstudio.microsoft.com/es/vs/)  
  (Workload **ASP.NET y desarrollo web**)
- ✅ [.NET SDK 8.0 o superior](https://dotnet.microsoft.com/es-es/download)
- ✅ Navegador web (Edge, Chrome, etc.)

---------------------------------------------------------------------------------------------------------------------------------------

Cómo abrir y ejecutar el proyecto 

1. Clonar el repositorio desde GitHub

git clone https://github.com/GastonGalvez/eventhub-api.git

2. Abrir Visual Studio 2022
Ir a Archivo → Abrir → Proyecto/Solución
Seleccionar el archivo: EventHub.Api.sln

3. Compilar el proyecto
En la barra superior, asegurarse de que esté seleccionado el perfil: IIS Express  (o EventHub.Api)
Luego presionar F5 o hacer clic en el signo PLAY

4. Ver la documentación Swagger
Al ejecutarse se abrirá automáticamente el navegador con Swagger: http://localhost:xxxx/swagger
Desde allí podrá probar todos los endpoints
